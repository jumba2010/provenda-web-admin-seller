INTRODUCTION
------------
Sellers module for online sales platform, made with react and redux using firebase as backend

REQUIREMENTS
------------
* Nodejs 8 or higher

MAINTAINERS
-----------
Current maintainers:
 * Judiao Mbaua (https://www.linkedin.com/in/judiao-mbaua-56b39946/)


## Environment Prepare

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

### Start project

```bash
npm start
```

### Build project

```bash
npm run build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### Test code

```bash
npm test
```
